## 문제 설명
두 수를 입력받아 두 수의 최대공약수와 최소공배수를 반환하는 함수, solution을 완성해 보세요. 배열의 맨 앞에 최대공약수, 그다음 최소공배수를 넣어 반환하면 됩니다. 예를 들어 두 수 3, 12의 최대공약수는 3, 최소공배수는 12이므로 solution(3, 12)는 [3, 12]를 반환해야 합니다.

`제한 사항`

1. 두 수는 1이상 1000000이하의 자연수입니다.

`입출력 예`

|   n   |   m   | return  |
| :---: | :---: | :-----: |
|   3   |  12   | [3, 12] |
|   2   |   5   | [1, 10] |

`입출력 예 설명`

`입출력 예 #1`
위의 설명과 같습니다.

`입출력 예 #2`
자연수 2와 5의 최대공약수는 1, 최소공배수는 10이므로 [1, 10]을 리턴해야 합니다.

두 수의 최대공약수란 
$$n, m$$
의 가장 큰 공약수를 의미합니다.
따라서 둘 중에 큰 수를 기준으로 -1씩 감소시키며 두 수의 나머지가 모두 0이 되는 값, $$d$$를 찾아야 합니다. 
두 수를 나누었을 때 나머지를 모두 0으로 만드는 $$d$$를 두 수의 **최대공약수**라고 정의합니다.

최대공약수로 나누어진 두 수 $$n ,m$$은 **서로소**이며, 따라서 **최소공배수** $$p$$는 다음과 같이 정의할 수 있습니다.

 $$p = d \times n \times m$$  

~~~javascript
function solution(n, m) {
    const num = n > m ? n : m;  // 간단한 조건문
                                // 둘 중 큰 값을 반환
    var answer = [];
    for(i = num; i >= 1; i--){
        if(n % i + m % i == 0){ // 나머지가 둘 다 0
            answer.push(i);     // 최대공약수 저장
            break;              // for문 탈출
        }
    }
    answer.push(n * m / i)      // 최소공배수
    return answer;
}
~~~
